1. Неоптимізований запит

Початковий запит побудувала з використанням вкладених підзапитів для підрахунку загальної кількості замовлень клієнта та кількості замовлень по категоріях товарів

Час виконання: ~406ms

2. Оптимізований запит

Для покращення продуктивності створила індекси для колонок, замість вкладених підзапитів використала СТЕ для єдиного проходу по таблицях та віконні функції для розрахунку статистики без втрати структури рядків, для видалення дублікатів застосувала оператор distinct

Час виконання: ~233ms

<img width="2650" height="848" alt="image" src="https://github.com/user-attachments/assets/a5ca7ba8-4bb7-4915-8d44-68519b1c26b9" />

3. Додаткове завдання: Optimizer Control

Для демонстрації ручного керування планувальником запитів тимчасово вимкнула використання індексів за допомогою:
SET enable_indexscan = off;
SET enable_bitmapscan = off;

Час виконання: ~172ms
